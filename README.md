# Playground.py

A Python learning repository for experiments, practice problems, and interview preparation.

This is a personal sandbox used to explore Python concepts, test ideas, and build fluency through hands-on work.

## Purpose

This repository exists to support learning by doing. It includes a mix of structured practice and unstructured experimentation, with an emphasis on clarity, iteration, and improvement over time.

---

## Contents

- **Python fundamentals**
  - Syntax, control flow, data structures
  - Functions, modules, and object-oriented programming

- **Problem solving**
  - Algorithms and data structures
  - Common interview-style questions
  - Edge cases and optimizations

- **Experiments**
  - Small scripts and utilities
  - Concept tests and quick prototypes

- **Notes**
  - Inline comments
  - Learning references and reminders

---

## Intended Audience

- Python learners at beginner to intermediate levels
- Developers refreshing core Python skills
- Anyone preparing for technical interviews

---

## Usage

Clone the repository and run any file directly with Python. Files are largely self-contained and meant to be read, modified, and refactored as learning progresses.

## Status

Ongoing and evolving. Code quality may vary as ideas are tested and revisited.
